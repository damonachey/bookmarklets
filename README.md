# bookmarklets

A collection of bookmarklets for quick link copying in various formats.

## Available Bookmarklets

### Markdown Copy Link
Copies the current page title and URL as a Markdown link format: `[Title](URL)`

**Usage**: Drag the "Markdown Copy Link.js" to your bookmarks bar, then click it on any page to copy a Markdown-formatted link to your clipboard.

### ZimWiki Copy Link
Copies the current page title and URL as a ZimWiki link format: `[[URL|Title]]`

**Usage**: Drag the "ZimWiki Copy Link.js" to your bookmarks bar, then click it on any page to copy a ZimWiki-formatted link to your clipboard.

## Installation
1. Right-click on each `.js` file and select "Copy Link Address" or similar
2. Edit the bookmark in your browser and paste the JavaScript code
3. Alternatively, drag the `.js` files directly to your bookmarks bar

## Compatibility
These bookmarklets work in modern browsers that support the Clipboard API and `document.execCommand('copy')`.
